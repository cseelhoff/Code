Run a **grilling** session, using the **domain-modeling** skill as well.

That means: interview one question at a time (grilling), and while decisions
crystallise, challenge domain language, update `CONTEXT.md`, and offer ADRs
when warranted (domain-modeling). Format files live under the
`domain-modeling` skill folder.

Do not enact the plan until the user confirms shared understanding.
